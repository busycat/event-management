# Getting Started
## Clone the repo
```
git clone https://github.com/cossth/event-management.git
```
## Change Connection string in `src/app.ts:16`
```
export const connectionString = "mongodb://usr:pswd@localhost/admin";
```
## Run
Enter the following commands
```
npm install
npm run watch
```
